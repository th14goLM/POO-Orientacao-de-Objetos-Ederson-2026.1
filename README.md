# 📚 Repositório de Programação Orientada a Objetos (POO)

Este repositório foi criado com o objetivo de armazenar **atividades, exercícios e projetos** desenvolvidos durante os estudos de **Programação Orientada a Objetos (POO)**.

---

## 🚀 Conteúdo

O repositório está organizado em pastas, contendo:

* 📌 **Exercícios básicos**

  * Classes e objetos
  * Atributos e métodos
  * Construtores

* 📌 **Conceitos fundamentais**

  * Encapsulamento
  * Herança
  * Polimorfismo
  * Abstração

* 📌 **Projetos práticos**

  * Aulas de aprendizagem 
  * Sistemas simulados (ex: foguetes, satélites, controle de missão)
  * Aplicações com menu interativo
  * Integração entre múltiplas classes

---

## 🛠️ Tecnologias utilizadas

* Java ☕
* Programação Orientada a Objetos
* Estruturas de dados básicas
